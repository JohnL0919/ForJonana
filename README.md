# ForJonana - Link coming soon...

An interactive web application featuring a playful subscription renewal interface with an escaping "No" button, background video, and success celebration page.

## Purpose
A lighthearted relationship subscription renewal app that makes the "No" button run away when users try to click it, leading to an inevitable "Yes" with celebratory results.

## Tech Stack
- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **YouTube IFrame API** - Background video
- **Spotify Web API** - Music integration
